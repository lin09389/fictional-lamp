羽毛球动作分析系统 - 使用说明
==============================

系统要求:
- Windows 7/8/10/11
- Python 3.7 或更高版本
- 至少 2GB 可用磁盘空间
- 摄像头设备

打包步骤:
1. 确保所有文件在同一目录下
2. 双击运行 build_app.bat
3. 等待打包完成（10-20分钟）
4. 在 dist/羽毛球动作分析系统/ 目录下找到可执行文件

运行程序:
1. 进入 dist/羽毛球动作分析系统/ 目录
2. 双击 "羽毛球动作分析系统.exe" 运行

注意事项:
- 首次打包需要较长时间，请耐心等待
- 打包过程中请勿关闭命令行窗口
- 建议关闭杀毒软件以提高打包速度
- 确保有足够的系统资源

技术支持:
如果遇到问题，请检查:
1. Python是否正确安装并添加到PATH
2. 磁盘空间是否充足
3. 杀毒软件是否阻止了打包过程
